I am the Lead Deep Learning Scientist at the University of Illinois, where I develop machine learning methods for medical imaging and computational biology.

My current work centers on cross-modal medical image synthesis — generating PET images from MRI through magnetic resonance fingerprinting, using [3D U-Net and Fourier neural operator architectures](https://github.com/nahilsobh/mri-pet-synthesis) built on a [clinical-grade preprocessing pipeline](https://github.com/nahilsobh/mri-pet-preprocessing). I also work on implicit neural representations ([SIREN](https://github.com/nahilsobh/cosiren) variants for continuous field estimation) and [physics-informed neural networks](https://github.com/nahilsobh/pinns-bc) with hard-enforced boundary conditions. Previously I built genomics analysis pipelines for the [KnowEnG](https://github.com/KnowEnG) center, including tools for [phenotype prediction](https://github.com/nahilsobh/phenotype-prediction-pipeline) and [gene regulatory network reconstruction](https://github.com/nahilsobh/gene-regulatory-network-reconstruction).
